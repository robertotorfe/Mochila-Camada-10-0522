¿Qué función cumple el hypervisor en la virtualización?
es un software que crea y ejecuta máquinas virtuales (VM) y que, además, aísla su sistema operativo 
y recursos de las máquinas virtuales y permite crearlas y gestionarlas.
Cuando el sistema de hardware físico se usa como hipervisor, se denomina "host", y las múltiples máquinas virtuales
que utilizan sus recursos se denominan "guests".
El hipervisor utiliza los recursos, como la CPU, la memoria y el almacenamiento, como un conjunto de medios que 
pueden redistribuirse fácilmente entre los guests actuales o en las máquinas virtuales nuevas.
