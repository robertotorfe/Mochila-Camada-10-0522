1)Git es un software de control de versiones el cual su objetivo es crear mayor eficiencia, confiabilidad y compatibilidad del mantenimiento de versiones. El proposito principal es llevar un registro de todos los cambios que realizan un equipo entero de desarrolladores que trabajan en 1 repositorio compartido.
2)En la actualidad el dueño de Github es Microsoft, al realizar la compra se planteo que se cambio la idea de enemigo del opensourse, ahora ellos colaboran al 100%. Microsoft se ha convertido en el principal colaborador de Linux y Kubernetes, desarrolla y distribuye productos basados ​​en Linux, ha abierto .NET, y son patrocinadores de The Linux Foundation, Apache Software Foundation, Open Source Initiative y muchos esfuerzos similares 
3)Si, hay muchas IDE que integran github en su estructura. Ej, Eclipse, VSC, PyCharm, Atom, etc
