¿Quién inventó el sistema de control de versión Git y por qué?

diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones

¿A quién pertenece actualmente Github y por qué?

Actualmente le pertenece a Microsoft que lo adquirio en 2018, pensando en un destino para que los desarrolladores aprendan, compartan y trabajen juntos para crear software
